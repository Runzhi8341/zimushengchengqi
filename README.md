# 图片字幕生成器

一个轻量级网页工具，用来给图片快速添加底部多行字幕条。适合自媒体配图、观点海报、金句图、活动记录图和社交平台传播图。

![图片字幕生成器界面](文字输入器.png)

## 项目特点

- 本地上传图片，支持 JPG、PNG、WEBP
- 多行字幕输入，每一行都会生成独立字幕条
- 底部字幕区域带“分层切割感”，适合短视频和社媒配图
- 支持调整字幕高度、字体大小、字体颜色、描边颜色、背景颜色和透明度
- 支持字体样式、字体粗细配置
- 自动预览生成效果
- 支持导出 PNG 图片
- 纯前端实现，不需要后端服务

## 使用方式

下载项目后，直接用浏览器打开 `index.html` 即可使用。

也可以用本地静态服务打开：

```bash
python -m http.server 8000
```

然后在浏览器访问：

```text
http://localhost:8000
```

## 操作流程

1. 点击“选择图片”，上传一张本地图片。
2. 在“字幕内容”里输入多行文字，每一行对应一条字幕。
3. 按需调整字幕高度、字体大小、颜色、背景透明度等参数。
4. 点击“生成字幕图片”查看最终效果。
5. 点击“保存图片”，导出 PNG 文件。

## 核心文件说明

- `index.html`：页面结构，包含上传区、参数设置区、字幕输入区和预览区。
- `styles.css`：页面样式。
- `script.js`：图片读取、Canvas 渲染、字幕生成和图片导出逻辑。
- `prd.md`：产品需求文档，记录工具设计目标和字幕渲染规则。

## 字幕渲染规则

- 输入框中的每一行文字都会被当作一条独立字幕。
- 每条字幕都有独立背景条，不共用一整块背景蒙层。
- 多条字幕会堆叠在图片底部，形成清晰的横向切割感。
- 字幕文本默认居中显示，并带描边增强可读性。
- 当字幕过长时，会自动缩小字号，仍然过长则使用省略号处理。
